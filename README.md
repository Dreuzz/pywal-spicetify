# pywal-spicetify
A simple cli tool for applying wal generated colors to spicetify.

## Installation

```
From Source
```bash
cargo build --release
```
## Usage
```bash
Usage: pywal-spicetify [OPTIONS] <theme>

Arguments:
  <theme>

Options:
  -r, --reset
  -h, --help     Print help
  -V, --version  Print version
```
